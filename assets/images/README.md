# Изображения для Missing Number

## leo-head.png

**Требуется:** Сохраните изображение львенка как `leo-head.png` в эту папку.

**Характеристики:**
- Формат: PNG
- Фон: прозрачный
- Рекомендуемый размер: 200x200 пикселей или больше
- Используется:
  - В шапке сайта (логотип)
  - В уравнениях (вместо неизвестного числа)

**Путь:** `/assets/images/leo-head.png`

---

## Инструкция по добавлению изображения:

1. Сохраните изображение львенка из сообщения
2. Переименуйте файл в `leo-head.png`
3. Поместите его в папку `assets/images/`
4. Убедитесь, что фон прозрачный (PNG с альфа-каналом)
5. Удалите или переименуйте старый `leo-head.svg` (опционально)

После добавления изображения тренажер будет использовать его вместо SVG.
